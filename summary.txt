The LangChain Team has released the LangSmith CLI and a set of "skills" to equip AI coding agents with expertise in the LangSmith ecosystem, focusing on tracing, execution analysis, test set building, and performance evaluation. This initiative significantly improved Claude Code's performance on these tasks, raising its pass rate from **17% to 92%** on their evaluation set.

The LangSmith CLI serves as an agent-native tool, enabling both agents and developers to interact with LangSmith functions like fetching traces, curating datasets, and running experiments directly from the terminal. It can be installed with the command `curl -sSL https://raw.githubusercontent.com/langchain-ai/langsmith-cli/main/scripts/install.sh | sh`. Skills are curated instructions, scripts, and resources that enhance agent performance by dynamically loading through progressive disclosure, thus preventing the performance degradation associated with providing too many tools. These skills are portable markdown files and scripts.

Within the `langsmith-skills` repository, three initial core skills are maintained: `trace` for adding tracing to code and querying traces, `dataset` for building example datasets, and `evaluator` for evaluating agents over those datasets. These skills facilitate a virtuous cycle in agent development, allowing agents to add tracing logic, debug behavior, create systematic testing datasets, and iterate based on evaluations and human feedback. Skills can be installed locally using `npx skills add langchain-ai/langsmith-skills --skill '*' --yes`, globally with `npx skills add langchain-ai/langsmith-skills --skill '*' --yes --global`, or linked to specific agents like Claude Code using `npx skills add langchain-ai/langsmith-skills --agent claude-code --skill '*' --yes --global`. The team plans to expand the skill set, including new capabilities for LangChain, LangGraph, and DeepAgents.
