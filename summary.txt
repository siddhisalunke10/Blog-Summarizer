The LangChain team has introduced the LangSmith CLI and a foundational set of skills to empower AI coding agents within the LangSmith ecosystem. This release enhances agent capabilities for adding tracing, understanding execution, building test sets, and evaluating performance. Notably, the implementation of these skills dramatically improved Claude Code (Sonnet 4.6)'s performance on LangSmith tasks, increasing its pass rate from 17% to 92% on their evaluation set.

The LangSmith CLI serves as an agent-native tool, enabling agents and developers to fetch traces, curate datasets, and run experiments seamlessly. It can be installed using the command `curl -sSL https://raw.githubusercontent.com/langchain-ai/langsmith-cli/main/scripts/install.sh | sh`. Skills are dynamically loaded instructions, scripts, and resources designed to enhance agent performance in specialized domains without causing degradation from too many tools. The initial set of LangSmith skills, housed in the `langsmith-skills` repository, comprises three key functionalities: `trace` for adding tracing to code and querying traces, `dataset` for building example datasets, and `evaluator` for assessing agents.

These skills can be installed locally with `npx skills add langchain-ai/langsmith-skills --skill '*' --yes`, globally with `npx skills add langchain-ai/langsmith-skills --skill '*' --yes --global`, or linked to a specific agent like Claude Code using `npx skills add langchain-ai/langsmith-skills --agent claude-code --skill '*' --yes --global`. This framework establishes a virtuous cycle for agent development, allowing agents to manage tracing, debug behavior, create systematic testing datasets, run evaluations, and iterate on their architecture. The team plans to continually expand the skill content and release skills for LangChain's open-source libraries, including LangChain, LangGraph, and DeepAgents.
